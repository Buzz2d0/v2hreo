Swift 联动 CGO 开发的 V2ray MacOS 菜单栏应用。

---

支持的功能:
- **默认配置:** `socks://127.0.0.1:1080` 目前还不支持修改
- **订阅地址:** `vmess://*` 或者 `http://*`(返回也的是`vmess://`)
- **服务器选择:** 
    - Load: 从订阅地址获取服务器列表
    - Ping: 获取连接服务器延迟
    - 下拉选择代理的服务器
